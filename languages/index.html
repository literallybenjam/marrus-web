Languages

I make languages (the kind you can speak)! My latest task has been the composition of Classical Sevensi: A Primer, documenting the grammars and providing an introductory lexicon for the Classical Sevensi language, which is underway. In the process of language development, I have also written technical standards such as that for LexisML Index Records or for encoding LANGDEV into Unicode, and programmed web technologies for processing and handling that information. Information on all of these efforts is available through the LANGDEV Project GitHub repo, where all the language data is stored. You can also check out the LANGDEV dictionaries to get a preliminary sense for the languages' current development status.
